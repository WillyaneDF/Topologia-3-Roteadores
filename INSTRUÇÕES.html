Você deve configurar a topologia de modo que as TRÊS REDES se comuniquem.

A rede mãe 172.16.0.0/16 deve ser utilizada e distribuida por todas as VLANs de todas as redes conforme a necessidade.

Todos os dispositivos tem que se comunicar com todos os dispositivos e todos eles devem acessar TODOS os sites criados nos servidores.

Todos os registros de DNS devem ser do tipo A (A RECORDS)

 REDE 1

Deve-se criar três VLANs

VLAN NOME Nº DE HOSTS
10 USUARIOS1 65
20 TI1 30
30 SERVIDORES1 4


Deve-se seguir a seguinte tabela de roteamento

DISPOSITIVO INTERFACE IP
RT-01 g0/0.10 1º IP VÁLIDO DA REDE
RT-01 g0/0.20 1º IP VÁLIDO DA REDE
RT-01 g0/0.30 1º IP VÁLIDO DA REDE
SRV-01 fa0 2º IP VÁLIDO DA REDE

O PC-01 e PC-02 devem receber as informações via DHCP.

Para o serviço de DHCP, você deve criar os seguintes escopos:

NOME DO ESCOPO: VLAN10

  • DEFAULT-GATEWAY: 1º IP VÁLIDO DA REDE
  • DNS Server: IP DO SRV-01
  • Start IP Address: 2º IP VÁLIDO DA REDE
  • SUBNET MASK: MÁSCARA DA VLAN 10
  • Número máximo de usuários: 10

NOME DO ESCOPO: VLAN20

  • DEFAULT-GATEWAY: 1º IP VÁLIDO DA REDE
  • DNS Server: IP DO SRV-01
  • Start IP Address: 2º IP VÁLIDO DA REDE
  • SUBNET MASK: MÁSCARA DA VLAN 20
  • Número máximo de usuários: 10

Criar o site www.rede1.com.br no SRV-01, o qual deve exibir a mensagem "BEM VINDO AO SITE DA REDE 1".
